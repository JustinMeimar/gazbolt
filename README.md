## GazBolt

In Fall of 2024, CMPUT 415 hosted a fork of Compiler Explorer. Hosting this fork had some complexities. First, there is an `npm build` step which required ~`4GB` of RAM. This pushed our server options into expensive territory. Second, [this file](https://github.com/compiler-explorer/compiler-explorer/blob/main/lib/base-compiler.ts). 


`Gazbolt` was formulated to simplify the deployment and development processes by creating a standalone, tailored implementation.

`Gazbolt` is currently a work in progress.
